**Penguin Brothers Web App**

---

## 📄 OVERVIEW

The new Penguin Brothers web app will be a fully custom-built **Next.js** application designed to replace the current Webflow site and improve the customer experience across online ordering, catering bookings, and client communications. This application will provide a seamless, modern interface with real-time integrations to third-party services like Square (for payments) and GoHighLevel (for CRM).

The app includes:
- Online ordering with real-time prep and pickup scheduling
- Custom catering booking with availability, add-ons, and quotes
- CRM-integrated forms with customer tracking via GoHighLevel
- Admin-friendly content updates and scalable architecture
