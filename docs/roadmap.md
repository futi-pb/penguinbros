## 📊 ROADMAP

### Phase 1: Planning & Setup
- Requirements gathering and user flow mapping
- Wireframes for ordering and booking experiences
- Set up project with Next.js, TypeScript, TailwindCSS, Supabase

### Phase 2: Frontend Build
- Home + Location + Menu pages
- Online ordering builder UI
- Catering booking flow UI
- Responsive design + animations

### Phase 3: Backend Integration
- Supabase setup for user/order data
- Square API for payment processing
- GoHighLevel webhook integration (custom-built API route)
- Admin logic for catering schedule availability

### Phase 4: Testing & Launch
- QA + device testing
- Stakeholder review
- Deployment via Netlify
- Monitoring and analytics setup

### Phase 5: Optional Auth + Rewards System
- Enable Supabase Auth for customer accounts
- Add login/logout and session management
- Allow users to view past orders and save favorites
- Track loyalty progress (e.g. "10th pookie free")
- Personalized deals via GoHighLevel CRM automations
